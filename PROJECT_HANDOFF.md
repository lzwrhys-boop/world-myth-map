# 项目交接文档 · 世界神话地图

本文档供后续接手开发的同事或 Cursor Agent 快速了解项目状态与协作约定。**当前请勿再改功能代码时，请先通读本文件。**

---

## 1. 项目名称与用途

**项目名称：** 世界神话地图  

**项目用途：** 使用纯前端（HTML + CSS + JavaScript）与 3D 地球（Globe.gl），将世界各地神话、童话、民间传说与史诗类故事以地理点位形式可视化；支持搜索、分类筛选、国家统计与移动端浏览，并附带「关于项目」说明弹窗。

---

## 2. 当前文件结构

| 文件 | 说明 |
|------|------|
| `index.html` | 页面结构：顶栏、搜索框、地球区、底部分类、右侧故事卡片与排行榜、「关于项目」弹窗；脚本引入顺序为 `data.js` → `script.js` |
| `style.css` | 全局与组件样式、桌面两栏布局、`<768px` 手机端适配、弹窗与搜索框等视觉 |
| `script.js` | 地球初始化、自转暂停/恢复、点位渲染与点击、搜索与分类过滤、排行榜与故事卡片、关于弹窗开关 |
| `data.js` | 故事数据库：`window.STORIES` 数组，每条含 `cn`、`en`、`country`、`region`、`lat`、`lng`、`category`、`era`、`summary`、`sourceType`、`confidence` 等字段 |
| `README.md` | 面向使用者的项目简介与运行说明 |

其他：`README.md` 中若提到本地预览方式，以仓库内实际说明为准；静态部署可使用 GitHub Pages（仓库中如有 `.nojekyll` 则用于跳过 Jekyll）。

## 3. 当前已实现功能

- **3D 地球**：Globe.gl + Three.js（CDN），随容器尺寸变化调整画布大小  
- **神话故事点位**：发光圆点 HTML 标记，无 emoji 地图乱码问题  
- **自转交互**：默认缓慢自转；鼠标移入地球卡片区域暂停，移出恢复；拖拽时暂停，逻辑与地球区域绑定稳定  
- **点位点击**：点击后右侧故事卡片展示对应条目；选中高亮  
- **搜索功能**：支持在 `cn`、`en`、`country`、`region`、`category`、`era`、`sourceType`、`summary` 等字段中匹配；大小写与空格有一定容错  
- **中文国家名搜索映射**：常见中文/简称映射到 `data.js` 中的英文 `country`，便于中文检索  
- **分类筛选**：底部胶囊按钮，与搜索条件可叠加  
- **国家排行榜**：基于当前筛选 + 搜索结果动态统计，列表可滚动  
- **手机端适配**：窄屏下纵向排版、地球在上、故事与排行在中、分类横向滑动等  
- **关于项目弹窗**：顶栏「关于项目」按钮打开居中说明弹窗，遮罩/关闭键/Esc 关闭

## 4. 当前不能随便改动的地方

以下模块已经过多轮联调，**无充分理由请勿大改**，避免回归：

- **地球容器布局**（`.globe-panel`、`#globeViz`、与主栅格相关的结构）  
- **右侧栏布局**（固定宽度、与左侧等高、卡片与排行榜分区）  
- **点位点击逻辑**（HTML marker 的创建方式、避免 hover 时整表重绘导致闪烁/点不中）  
- **自转暂停逻辑**（绑定在 `.globe-panel` 等稳定容器上，勿绑在易变子节点上）  
- **搜索与筛选联动逻辑**（`getFilteredStories`、与地球点位、排行榜、故事卡片的一致性）  

若必须修改，建议先本地全量回归：地球显示、点击、搜索、筛选、排行榜、手机端、关于弹窗。

## 6. 给下一个 Cursor Agent 的开发注意事项

1. **大改前先阅读本文件**（及 `README.md`），确认改动范围与风险。  
2. **一次只改一个功能**，便于 diff 审查与回滚。  
3. **不要同时改布局、数据和交互**；若必须动多处，拆成多次提交或多次对话步骤。  
4. **修改数据优先改 `data.js`**，保持 `window.STORIES` 结构一致。  
5. **修改视觉优先改 `style.css`**，避免在 `script.js` 里堆样式字符串。  
6. **修改交互优先改 `script.js`**，避免在 HTML 内联脚本中分散逻辑。  
7. 改动与地球/点位相关时，务必在**桌面 + 手机宽度**下各测一遍点击与自转。
